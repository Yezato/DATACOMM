# How to Install OpenStack Using Kolla Ansible
## This tutorial utilizes the specifications below:
- OpenStack version: ZED
- OS : Ubuntu 22.04
- 2 network interfaces
- 8GB main memory
- 50GB disk space

## Service openstack:
- Placement (placement)
- Neutron (network)
- Nova (compute)
- Cinder (volume)
- Glance (image)
- Keystone (identity)
- Gnocchi (metric)
- Aodh (alarming)
- Ceilometer (metering)
- Horizon (Dashboard)
